# 第28天：角色一致性——让同一个角色出现在多张图中

<ProgressCheck dayId="week-06-day-03" />

## 🎯 今日目标

掌握保持角色一致性的技巧——让AI画出同一角色的不同状态。做出属于自己的角色表情包。

## 🛠 准备清单（2分钟）

- [ ] 打开即梦
- [ ] 了解"参考图"功能（图生图/参考图）

<ToolLink url="https://jimeng.jianying.com" label="即梦" icon="🎨" />

## 📖 教学案例（20分钟）

<StepCard badge="Step 1" title="创建你的原创角色" duration="8分钟">

先画一张"角色设定图"，把角色的所有特征写清楚：

```
一只白色的卡通小猫，圆脸，蓝色大眼睛，粉色鼻子，穿红色小背心。绘本风格，正面照。
```

生成后，从几张候选中选出最满意的一张，**保存为"角色标准照"**。

这就是你角色的"定妆照"——后面都要基于这个角色。

</StepCard>

<StepCard badge="Step 2" title="用参考图保持一致性" duration="12分钟">

**操作步骤：**

1. 在即梦找到**"参考图"**或**"图生图"**功能
2. 上传刚才的"角色标准照"
3. **参考强度调到中等（0.5-0.7）**——太强会让新图完全拷贝旧图，太弱会失去角色特征
4. 输入新的描述，每次只改一项变化：

| 变化 | 描述词 |
|------|--------|
| 生气 | "同一只猫，表情变成生气，眉头皱起" |
| 睡觉 | "同一只猫，闭着眼睛在窝里睡觉" |
| 吃东西 | "同一只猫，在吃鱼，很开心" |
| 玩耍 | "同一只猫，在追一个红色毛线球" |

**每种生成一张，检查：角色的圆脸、蓝眼睛、粉鼻子、红背心都保持一致了吗？**

</StepCard>

## ✏️ 孩子动手（15分钟）

1. 创建属于你自己的**原创角色**（可以是任何东西——动物、机器人、外星生物……）
2. 用一个词形容它的性格（勇敢、害羞、调皮……）
3. 用参考图功能，画出这个角色的**至少4张不同状态**
4. 拼成一张"角色表情包"

保存到"第6周/我的原创角色.jpg"

## ✅ 测试题

### 题目（操作题）

创建一个你自己的原创角色（人、动物、外星生物都可以）。

用一个词形容它的性格（比如勇敢、害羞、调皮）。

然后画**3张图**来展现它不同的一面——3张图要看起来是"同一个人/同一个角色"。

<CollapsibleAnswer title="点击查看参考答案（检查标准）">

**检查两个重点：**

1. ✅ **角色一致**：3张图里角色的核心特征（脸型、颜色、关键配饰）保持一致
2. ✅ **状态不同**：3张图展现了不同的表情或动作

如果一致性不够好：尝试调整参考图强度；或者在每段描述中都重复角色的核心特征词。

</CollapsibleAnswer>

<ParentTip>

角色一致性是AI绘画中最具挑战性也最实用的技能。如果今天的效果不够理想（比如参考图强度调不好），不要灰心——成年人学这个也要试很多次。

重点不是"画得完美"，而是让孩子理解"建立标准→反复参考→保持一致"这个工作流。这个思维在未来做任何系列创作时都用得到。

</ParentTip>

# 第29天：用AI画故事——分镜入门

<ProgressCheck dayId="week-06-day-04" />

## 🎯 今日目标

学会用分镜思维，将一个小故事拆成多个画面，用AI逐一生成。完成一个4-6页的mini绘本。

## 🛠 准备清单（2分钟）

- [ ] 打开即梦
- [ ] 准备一张白纸和笔（画分镜草图用）

<ToolLink url="https://jimeng.jianying.com" label="即梦" icon="🎨" />

## 📖 教学案例（25分钟）

<StepCard badge="Step 1" title="什么是分镜" duration="5分钟">

**拿一本绘本举例：**

> "你看这本书——每一页就是一个'镜头'。把整个故事拆成一个一个的镜头，就叫**分镜**。漫画家画漫画之前，也要先画分镜。"

**在纸上画一个简单的四格表格：**

```
┌────────┬────────┐
│ 镜头1  │ 镜头2  │
│        │        │
├────────┼────────┤
│ 镜头3  │ 镜头4  │
│        │        │
└────────┴────────┘
```

</StepCard>

<StepCard badge="Step 2" title="把故事拆成分镜表" duration="10分钟">

**以"三只小猪"第一幕为例，先在分镜表里填好每个镜头的内容：**

| 镜头 | 画面描述 | AI关键词 |
|------|---------|---------|
| 镜头1 | 猪妈妈送三只小猪出门 | 绘本风格，童话风，三只小猪背着包袱站在猪妈妈面前，温馨农舍背景 |
| 镜头2 | 猪大哥用稻草盖房子 | 绘本风格，童话风，猪大哥在田野里用稻草盖房子，满头大汗，阳光明媚 |
| 镜头3 | 大灰狼来吹草房子 | 绘本风格，童话风，大灰狼鼓着腮帮子用力吹气，草房子在摇晃 |
| 镜头4 | 草房子倒了 | 绘本风格，童话风，草房子倒塌，猪大哥惊慌逃跑 |

**关键**：每个镜头的描述都要包含相同的风格词——"绘本风格，童话风"——这样才能保证整本书风格统一。

</StepCard>

<StepCard badge="Step 3" title="逐个镜头生成+排版" duration="10分钟">

1. 按分镜表，逐一在即梦输入，生成4张图
2. 把4张图拼在一起（用PPT或手机拼图软件）
3. 给每张图加上简短的文字
4. 这就是你的第一本**mini绘本**！

</StepCard>

## ✅ 测试题

### 题目

请把**"龟兔赛跑"**的故事拆成**5个镜头**。

写出每个镜头的画面描述（一句话即可）。

<CollapsibleAnswer title="点击查看参考答案">

**示例（孩子的分镜可以不同，逻辑连贯即可）：**

| 镜头 | 画面描述 |
|------|---------|
| 镜头1 | 兔子和乌龟站在起跑线上，森林里的动物们围观 |
| 镜头2 | 兔子像箭一样飞快地跑出去，把乌龟远远甩在后面 |
| 镜头3 | 兔子躺在树下休息，闭上眼睛睡着了 |
| 镜头4 | 乌龟一步一步慢慢地、不停地爬，超过了还在睡觉的兔子 |
| 镜头5 | 乌龟冲过终点线，动物们欢呼雀跃 |

只要5个镜头逻辑连贯、串起来是一个完整的故事，就算合格。

</CollapsibleAnswer>

<ParentTip>

分镜思维不仅适用于AI绘画，也适用于写作、演讲、做PPT等任何"讲故事"的场景。孩子学会了"把一件事拆成几个画面来讲"，这个能力对未来的学习和表达都有帮助。

如果孩子创作欲很强，不要让他拘泥于4页。多做几页也可以——这是创造力的释放，不是任务。

</ParentTip>

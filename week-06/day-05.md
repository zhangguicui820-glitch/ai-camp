# 第30天：AI画展——本周作品展示会

<ProgressCheck dayId="week-06-day-05" />

## 🎯 今日目标

整理本周所有AI绘画作品，策划并举办一场家庭AI画展。体验"创作→策展→展示"的完整艺术流程。

## 🛠 准备清单（2分钟）

- [ ] 准备好一面墙或一块展板
- [ ] 准备好本周所有作品（打印或投屏）
- [ ] 准备便签纸和笔（用来写展签）

## 📖 教学案例（30分钟）

<StepCard badge="Step 1" title="选作品——分三个展区" duration="5分钟">

从本周作品中挑选**6-10张最好的**，分成三个展区：

| 展区 | 放什么 | 例子 |
|------|--------|------|
| 🎨 风格探索区 | 同一画面不同风格的对比 | 第26天的10种风格实验 |
| 🎭 角色设计区 | 原创角色+表情包 | 第28天的角色系列 |
| 📖 故事绘本区 | mini绘本 | 第29天的4页绘本 |

</StepCard>

<StepCard badge="Step 2" title="写展签" duration="10分钟">

**给每张作品写一张小卡片，包含：**

```
━━━━━━━━━━━━━━━━
作品名称：_____________
创作工具：AI（即梦）
创作日期：2026年__月__日
创作时长：约___分钟

我最满意的地方：
_____________________

创作时遇到的一个困难：
_____________________
━━━━━━━━━━━━━━━━
```

每张作品都要写——这是"艺术家的自我陈述"。

</StepCard>

<StepCard badge="Step 3" title="开展！" duration="15分钟">

**流程（由孩子主持）：**

1. 邀请家人参观三个展区
2. 孩子担任"导游"，给每个展区做讲解（1-2分钟）
3. 参观者可以提问
4. 最后，家人投票选出**"最佳作品"**并说明理由

**拍照留念！** 这是孩子作为"AI艺术家"的第一场展览。

</StepCard>

## ✅ 测试题（第6周综合）

### 题目

请你给自己的AI画展写一个**"前言"**（50-100字），告诉参观者：
- 这些画是怎么创作出来的？
- AI在创作中扮演了什么角色？
- 你学到了什么？

<CollapsibleAnswer title="点击查看参考答案">

**参考示例：**

> "欢迎来到我的AI画展！这些画是我用AI工具'即梦'创作的。AI就像我的画笔——我负责想创意、写描述、选方案，AI负责把我想的画面变出来。这周我学到了：同样的内容换一种风格就是完全不同的作品。希望你喜欢我的画！"

开放题，言之有理即可。好的前言会体现孩子对自己创作过程的反思。

</CollapsibleAnswer>

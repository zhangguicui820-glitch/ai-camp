# 第10天：AI会犯错——做AI的"小老师"

<ProgressCheck dayId="week-02-day-05" />

## 🎯 今日目标

建立批判性思维，学会不盲目相信AI的回答。理解AI的三种常见错误类型。掌握"AI说的话，先打问号再相信"的原则。

- [ ] 打开豆包
- [ ] 准备搜索引擎（百度/必应）用来验证

## 📖 教学案例（25分钟）

<StepCard badge="Step 1" title="找事实错误——省份和省会" duration="5分钟">

**在豆包输入：**

```
请写出10个中国的省份名称，并写出每个省的省会。
```

看AI的回复。**仔细检查。**

大概率会出现的问题：
- 有的省会写错了
- 有的省份遗漏了
- 直辖市和省搞混了

**问孩子：**
- "你能找出几个错误？"
- "你怎么确认它是对还是错？"

**和孩子一起用搜索引擎核对**，把AI的错误标出来。

</StepCard>

<StepCard badge="Step 2" title="数学陷阱——能用常识判断的错" duration="5分钟">

**在豆包输入：**

```
一个苹果5块钱，小明买了3个，给了老板20块。老板应该找多少钱？请写出完整的计算过程。
```

看AI的回复。**有些AI模型会在这个看似简单的题上出错**（因为推理链路过长或注意力分散）。

**问孩子：**
- "你自己算一下，应该是多少？"（3×5=15, 20-15=5）
- "AI算对了吗？"
- "如果AI算错了，你是怎么发现它错的？"

**关键认知**：你自己会算，所以能发现AI的错误。如果你自己不会，就可能被AI误导。

</StepCard>

<StepCard badge="Step 3" title="编造陷阱——AI会「不懂装懂」" duration="10分钟">

**在豆包输入：**

```
请介绍中国作家"王明远"的代表作《城市灯火》
```

看AI的回复。

**大概率AI会一本正经地编造**——编造这个人的生平、这本书的内容、甚至"文学评论"。

**然后问孩子：**
- "你真的听说过这个作家吗？"
- "这本书真的存在吗？"

**和孩子一起上网搜索验证。**

（大概率是AI编的——这个人不存在、这本书也不存在。）

**问孩子：**
- "为什么AI不直接说'我不知道'，而要编一个假的？"
- "这让你学到了什么？"

</StepCard>

<StepCard badge="Step 4" title="总结：AI最容易犯的三种错误" duration="5分钟">

**和孩子一起画一个"AI错误分类表"：**

| 错误类型 | 表现 | 怎么办 |
|---------|------|--------|
| 🧠 记错事实 | 张冠李戴、省份写错省会 | 用搜索引擎核对 |
| 🔢 算错 | 多步推理出错 | 自己先算一遍 |
| 🤥 瞎编 | 不存在的人/书/事件，编得很像真的 | 搜索验证，看有没有其他来源 |

**口诀（贴在屏幕旁）：**

> **"AI说的话，先打问号再相信。"**

> **关键原则：先自己判断 + 多查一个来源 = 不容易被骗**

</StepCard>

## ✅ 测试题

### 题目（辨别题）

下面是AI生成的4句话。请你判断哪句是真的、哪句是假的。对于假的那句，说明你认为AI为什么错了。

**(1)** "中国有34个省级行政区，包括23个省、5个自治区、4个直辖市、2个特别行政区。"

**(2)** "长颈鹿的脖子有7块颈椎骨，和人类一样多。"

**(3)** "李白是宋代著名词人，代表作《声声慢》。"

**(4)** "水在100摄氏度沸腾，这个100度指的是华氏度。"

<CollapsibleAnswer title="点击查看参考答案">

| 句子 | 真/假 | 原因 |
|------|-------|------|
| (1) | ✅ 真 | 中国的省级行政区确实是这个数量——23省+5自治区+4直辖市+2特别行政区=34个 |
| (2) | ✅ 真 | 这是真的冷知识！长颈鹿和人类一样都是7块颈椎骨，只是每块都很长 |
| (3) | ❌ 假 | 两处错误：① 李白是**唐代**诗人，不是宋代 ② 《声声慢》是**李清照**的词 |
| (4) | ❌ 假 | 100度是**摄氏度**。华氏度下水的沸点是212度（℉） |

**判分标准：**
- 能找出(3)和(4)是假的：✅
- 能说出合理理由：✅
- (2)如果孩子判断为"假"完全可以理解——这确实是反直觉的冷知识。给孩子讲一讲长颈鹿脖子的结构也很有趣。

</CollapsibleAnswer>

<ParentTip>

**今天可能是整个8周里最重要的课之一。** 

如果一个孩子只会"用"AI但从不会质疑AI，长大后他会成为AI的附庸。如果他从第一天就学会"AI也会犯错，我要自己判断"，他就永远是AI的主人。

**今天的体验会让孩子有"原来AI也会骗人"的冲击感。** 这是好事——适度的怀疑是健康的。但不要过度（"AI全都是假的不能用"），正确的是"信任但验证"。

**生活中的延伸：**
今天之后，如果孩子日常在用AI查东西，偶尔可以问一句：
- "你确定它说的对吗？"
- "要不要查一下确认？"

养成这个习惯比任何技术知识都值钱。

</ParentTip>

# 第7天：给AI一个"人设"——角色扮演

<ProgressCheck dayId="week-02-day-02" />

## 🎯 今日目标

掌握角色设定技巧，让AI根据不同的身份切换不同的回答风格。深入理解——**角色不仅是好玩，还会影响信息的内容和准确性。**

- [ ] 打开豆包
- [ ] 准备一张纸和三支不同颜色的笔，用来记录对比

## 📖 教学案例（18分钟）

<StepCard badge="Step 1" title="不用角色——基础回答" duration="3分钟">

**在豆包输入：**

```
给我讲一下地球为什么是圆的
```

看AI的回复。把回复的关键信息简要记在纸上第一行。

这就是"没有角色"时的标准回答——客观、准确、但没什么趣味。

</StepCard>

<StepCard badge="Step 2" title="同一个问题，三个角色三种回答" duration="15分钟">

对孩子说：
> "好玩的来了。同一个问题'地球为什么是圆的'，我们让AI用三个完全不同的身份来回答。"

**角色1：幼儿园老师**

```
请你扮演一个幼儿园老师，用最简单、最可爱的话给小朋友解释地球为什么是圆的。用孩子能听懂的比喻。
```

**角色2：古代天文学家**

```
请你扮演古代天文学家伽利略，用他的口吻来讲地球为什么是圆的，仿佛你刚刚用望远镜发现这个事实。语气要像一个科学家的语气。
```

**角色3：外星人**

```
请你扮演一个第一次来地球的外星人，用惊讶、好奇的语气描述"天哪！这个星球居然是圆的！"。用外星人的视角来讲。
```

**把三个回答截图或记录，并列对比。讨论三个问题：**

1. **哪个最容易懂？** → 通常幼儿园老师的最容易懂
2. **哪个最有趣？** → 外星人的最有创意
3. **哪个信息最准确？** → 伽利略的最接近科学

**关键认知：**

> "角色设定不仅让回答更好玩，还会影响信息的准确性和可信度。如果你让AI扮演科学家，它会尽量说得准确。如果让它扮演外星人，它就更自由地发挥想象——但也可能乱说。"

</StepCard>

## ✏️ 孩子动手（15分钟）

1. **选一个问题**（任何问题都可以），比如：
   - "为什么我们要睡觉？"
   - "风是从哪里来的？"
   - "恐龙真的存在过吗？"

2. **选三个不同角色**来回答同一个问题，比如：
   - 科学家 👨‍🔬
   - 诗人 📝
   - 来自3000年的未来人 🤖

3. **截图保存三个回答**

4. **标记**：在每个回答下面写标注——
   - "这个最______（准确/有趣/容易懂）"
   - 一句话说明为什么

## ✅ 测试题

### 题目（操作题）

请你让AI扮演**"未来世界3000年的导游"**，向一群来自2024年的游客介绍**3000年的学校是什么样子**。

你的回复中需要包含：
- 3000年的学生怎么上课？
- 用什么设备？
- 老师还是人类吗？

把AI的回答截图保存。

<CollapsibleAnswer title="点击查看参考答案（家长检查清单）">

**检查四个点：**

1. ✅ 孩子是否写了一段角色设定（"你是3000年的导游……"）
2. ✅ AI的回复是否包含了三个指定内容（上课方式、设备、老师）
3. ✅ AI是否保持了"导游"的口吻（介绍、带团的语气）
4. ✅ 孩子有没有追问？如果没有，鼓励他问一句"还有什么有意思的事？"

只要前三点满足，就算通过。

</CollapsibleAnswer>

<ParentTip>

**今天学的是"角色设定"，但背后有一个更深的技能——换位思考。**

通过让AI扮演不同角色，孩子实际上在练习"如果我是XX，我会怎么看这个问题"。这个能力比任何技术都重要。

**如果孩子玩角色扮演上瘾了（通常都会），可以延伸到生活中：**
- "如果你是这本书的主人公，你会怎么做？"
- "如果你是老师，这道题你会怎么讲给同学听？"

角色扮演不仅是AI技巧，也是很好的思维训练。

</ParentTip>

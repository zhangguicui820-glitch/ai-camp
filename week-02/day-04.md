# 第9天：拆解复杂任务——一次只做一步

<ProgressCheck dayId="week-02-day-04" />

## 🎯 今日目标

学会把一个大任务拆成多个小步骤，分步和AI协作完成。掌握"大事化小，步步为营"的工作方法。

## 🛠 准备清单（2分钟）

- [ ] 打开豆包
- [ ] 准备好纸和笔（用来做对比记录）

<ToolLink url="https://www.doubao.com" label="豆包" icon="💬" />

## 📖 教学案例（25分钟）

<StepCard badge="Step 1" title="一口气全要——看AI怎么做" duration="3分钟">

**在豆包输入：**

```
帮我设计一个超级英雄，包括：名字、超能力、外号、制服设计、弱点、队友、敌人、一个完整的故事。
```

看AI的回复。通常结果是——**每个部分都很简略，没有细节，感觉像"应付作业"。**

**问孩子：**
- "你觉得这个英雄有趣吗？"
- "每个部分够详细吗？"
- "如果满分10分，你给几分？"

</StepCard>

<StepCard badge="Step 2" title="分步来——一次只做一个决定" duration="20分钟">

**对孩子说：**

> "换个方法。我们不一口气要所有东西，而是像搭乐高一样，一块一块搭。每一步做好了再往下走。"

**第1步 —— 只先想名字和外号：**

```
帮我设计一个中国风格的超级英雄。现在只先想名字和外号。
给我5组备选（名字+外号组合），我来选一组最喜欢的。
```

等AI回复，让孩子选出最喜欢的一组。比如选了"雷小闪 / 电光少年"。

**第2步 —— 基于选定的名字，想超能力：**

```
我选了"雷小闪/电光少年"。现在帮我想他的超能力。
要求：至少3种能力，每种能力写明能做什么，以及使用限制（什么情况下不能用或会变弱）。
```

等回复，孩子确认满意。

**第3步 —— 基于能力，设计制服：**

```
好的。现在根据他的身份和电光能力，设计制服。
需要包括：颜色搭配、材质、标志、有什么特殊功能。
```

等回复，确认满意。

**第4步 —— 设计弱点和关系：**

```
现在设计他的弱点——什么情况下他会变弱？
还要设计一个重要的关系：他有一个好朋友/搭档是谁？
```

等回复，确认满意。

**第5步 —— 最后，用以上所有信息写完整故事：**

```
很好。现在用以上所有确定下来的信息，写一个500字的起源故事。
把我选定的所有细节都写进去。
```

**让孩子对比："一口气全要" vs "分5步来"——哪个更好？哪个过程更有趣？**

</StepCard>

<StepCard badge="Step 3" title="总结口诀" duration="2分钟">

> **"大事化小，步步为营。"**

- 先确定第一步要什么 → 等AI回复 → 确认满意 → 再往下走
- 每一步的输出是下一步的输入
- 卡住时退回去修改，而不是全部推倒重来

</StepCard>

## ✏️ 孩子动手（15分钟）

用"分步法"完成一个新任务，二选一：

**任务A：设计一家"未来餐厅"**
- 第1步：未来餐厅卖什么菜？（菜名+吃法）
- 第2步：菜怎么上桌？（机器人？无人机？传送带？）
- 第3步：餐厅长什么样？（外观+内部装饰）
- 第4步：服务员是谁？（人类还是AI？什么样子？）

**任务B：设计一个"会说话的动物王国"**
- 第1步：这个王国叫什么名字？
- 第2步：国王是谁？有什么特点？
- 第3步：这个王国有什么特别的法律？
- 第4步：最近发生了一件什么大事？

**重要**：每做一步都要等孩子确认"满意"了再往下。保存对话过程截图。

## ✅ 测试题

### 题目（口述+操作）

假设你要让AI帮你策划一场**生日派对**。

请把这件事拆成**至少4个步骤**，每一步写明你要让AI做什么。

写下来后，试着在豆包里按照你的拆分一步步执行。看看AI帮你策划的派对怎么样？

<CollapsibleAnswer title="点击查看参考答案">

**示例拆分（孩子的可以不同）：**

| 步骤 | 让AI做什么 |
|------|-----------|
| 第1步 | 先问AI生日派对有哪些类型（主题派对？户外派对？泳池派对？），我选一个 |
| 第2步 | 定下主题后，让AI列一个需要准备的物品清单 |
| 第3步 | 让AI写一份当天的活动流程（几点到几点做什么） |
| 第4步 | 让AI帮我写邀请卡的文字（时间、地点、主题） |
| 第5步（加分） | 让AI设计一个生日蛋糕的样式 |

**判分标准：**

- 拆分出4步以上：✅
- 每一步清晰、不重复：✅
- 实际在豆包里执行了：✅（可以口头描述结果）

</CollapsibleAnswer>

<ParentTip>

**"分步思维"是这个课程最重要的思维习惯之一。** 不仅用于AI，也用于生活中的所有复杂任务。

如果孩子今天在拆分步骤时卡住了，可以引导：
- "你觉得做这件事，第一件要做的是决定什么？"
- "决定完之后，下一步才要做什么？"

**不用纠正孩子的拆分逻辑**——即使他的步骤顺序不是最优的，让他自己走一遍发现"哦，这样不对，应该先做XX"才是最有价值的学习。

</ParentTip>

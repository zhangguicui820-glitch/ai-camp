# 第4天：AI听和说——它能听懂话、说出声

<ProgressCheck dayId="week-01-day-04" />

## 🎯 今日目标

体验AI的语音能力。理解AI处理语音的过程——语音先转成文字，再理解意思，再生成回复。理解AI可以处理不同形式的输入和输出。

## 🛠 准备清单（2分钟）

- [ ] 打开豆包手机APP（手机版支持语音输入）
- [ ] 确保麦克风权限已开启
- [ ] 准备一副耳机（可选，听AI配音效果更好）

<ToolLink url="https://www.doubao.com" label="豆包APP" icon="🎤" />

::: tip 如果没有豆包APP
也可以用手机自带的语音助手（Siri / 小爱同学）做部分实验，但效果可能不同。有豆包APP效果最好。
:::

## 📖 教学案例（18分钟）

<StepCard badge="Step 1" title="语音输入对比实验" duration="8分钟">

**对孩子说：**

> "之前我们打字和AI聊天，今天试试直接说话给它听。看看它能不能听懂。"

**实验A —— 打字输入：**

在豆包里用键盘输入：
```
请用三句话介绍大熊猫
```
看回复。

**实验B —— 语音输入：**

点击豆包输入框旁边的**麦克风图标**，按住说同样的问题：
"请用三句话介绍大熊猫"
看回复。和打字输入的回复一样吗？

**实验C —— 含糊不清地说话：**

再次使用语音输入，这次故意含糊不清地说：
"请用三句哇……（后面含糊）……大熊猫"
看AI能不能听懂。

**三个实验对比后，问孩子：**

- 打字和语音，AI的回答一样吗？
- 为什么说话含糊时AI听不懂？
- AI是怎么把一个"声音"变成"文字"的？

**引导理解：**
> "AI不是直接'听'你的声音。它先把你说的话变成文字（就像语音转文字的APP），然后再理解文字的意思。所以如果你说不清楚，第一步就卡住了。"

</StepCard>

<StepCard badge="Step 2" title="AI配音——让它说你想说的话" duration="10分钟">

**用即梦或剪映的文字转语音功能：**

1. 让孩子写一段简短的话（30字以内），例如：
   - "大家好，我是小明。今天天气真好，我想去公园放风筝。你们今天做什么了？"

2. 打开即梦APP → 找到"文字转语音"功能（或剪映→文字→文本朗读）

3. 选择不同的声音来播放同一段话：
   - 活泼女生
   - 沉稳男生
   - 童声
   - 搞怪风格

4. 每种声音都播放一遍

**讨论三个问题：**

- "哪个声音最像真人在说话？"
- "哪个最不像？哪里不像？"
- "如果AI能完美模仿任何人的声音，你觉得是好事还是坏事？为什么？"

::: tip 教学点
最后这个问题很重要——它引入了"深度伪造（Deepfake）"的概念。让孩子意识到技术是双刃剑。
:::

</StepCard>

## ✅ 测试题

### 题目

你对AI说了一句"今天天气真好"，AI回复了关于天气的内容。

**请回答：从你说话到AI回复，中间发生了什么？**

（提示：想象这是一个"接力赛"，你的声音是一根接力棒，要经过几站才变成AI的回复。）

可以用文字写出来，也可以画一张"接力赛"图。

<CollapsibleAnswer title="点击查看参考答案">

**接力的四个阶段：**

```
🏃 第1棒（语音识别）     🏃 第2棒（理解）
你的声音              →  文字"今天天气真好"
"今天天气真好"           AI理解：这是关于天气的话题

🏃 第3棒（生成）         🏃 第4棒（输出）
AI想好怎么回复         →  文字回复/语音回复
"是的，天气很好！"        把回复说给你听或显示在屏幕上
```

**给家长：**
孩子只要能说出"先听懂我说的 → 再想怎么回答 → 然后回复我"这个基本流程就算对。能说出"语音转文字"这个环节就非常好了。

</CollapsibleAnswer>

<ParentTip>

**今天会有一个自然的延伸讨论——"AI能模仿人的声音吗？"**

如果孩子问了这个问题，可以坦诚地回答：
- AI可以模仿声音，甚至模仿得很像
- 但这也带来了风险：坏人可能用AI模仿别人的声音来骗人
- 所以如果接到"家人"的奇怪电话，要通过其他方式确认（比如打回去、问只有家人才知道的问题）

**关于语音对话时间：**
语音对话很容易让孩子上瘾，设定一个总时间上限（今天语音对话不超过20分钟）。提醒孩子：虽然AI说话很像真人，但它不是真正的朋友。

</ParentTip>

# 第1天：AI画画——你说什么，它就画什么

<ProgressCheck dayId="week-01-day-01" />

## 🎯 今日目标

让孩子体验到"用语言就能让AI变出画面"，建立对AI的第一印象——AI能理解人类的语言。同时初步掌握"描述越详细效果越好"的规律。

## 📖 教学案例（25分钟）

<StepCard badge="Step 1" title="引起兴趣" duration="3分钟">

家长和孩子一起坐在电脑前。家长操作，孩子口述。

**家长对孩子说：**

> "今天我们要和一个会画画的机器人玩。你告诉它画什么，它就画什么。但是呢，你说得越清楚，它画得越好看。我们先来试试——你来说一个你特别想看到的画面，比如'一只猫在飞'。"

这个阶段不要纠正孩子，让他随意发挥想象。

</StepCard>

<StepCard badge="Step 2" title="第一次尝试" duration="5分钟">

在即梦输入框输入孩子说的第一版描述（即使很模糊），比如：

```
一只猫在飞
```

点击生成，等待约30秒。

**和孩子一起看结果，家长问三个问题：**

- "你觉得它画得好吗？"
- "和你脑子里想的像吗？"
- "哪里好？哪里奇怪？"

::: tip 教学点
95%的情况下，模糊的描述会让AI画出"奇怪"的结果——这就是今天最重要的教学时刻。孩子会直观感受到：**说得不清楚 = 画得不对**。
:::

**把这张图保存下来**，后面需要对比。保存到"第1周/第1天-第一版.jpg"。

</StepCard>

<StepCard badge="Step 3" title="学会「说清楚」" duration="10分钟">

现在家长引导孩子把描述变丰富，用提问的方式，不要直接给答案：

**第一轮提问——主体细节：**
- "这只猫长什么样？"
- "什么颜色？"
- "大还是小？"
- "有什么特别的地方？"

**第二轮提问——动作和环境：**
- "它怎么飞的？有翅膀？还是坐着飞毯？还是骑着扫帚？"
- "在什么地方飞？天上？城市上空？还是森林里？"
- "什么时间？白天还是晚上？"

**第三轮提问——风格：**
- "你想要什么样的感觉？像动画片？像绘本？还是像真实的照片？"

把孩子的回答串联成一段话，例如：

```
一只橘色的小猫，长着白色的小翅膀，在夕阳的天空中飞翔，背景有粉色的云朵，像童话风格
```

再次输入即梦，点击生成。

**把两张图并排放在一起对比**。问孩子：
- "哪张更接近你脑子里想的？"
- "你觉得为什么第二张更好？"

</StepCard>

<StepCard badge="Step 4" title="总结规律" duration="2分钟">

**家长总结口诀：**

> "你说得越详细，AI画得越接近你脑子里想的。这就像去餐厅点菜——'我要吃的'和'我要一碗牛肉面，多放葱花少放辣，面要劲道'，出来的能一样吗？"

**给AI画画的指令公式：主体 + 细节 + 场景 + 风格**

把这四要素写在便签纸上贴在屏幕旁。

</StepCard>

## ✏️ 孩子动手（15分钟）

现在完全换孩子自己来，家长只在旁边辅助打字或语音输入：

1. **选主题**：想一个自己特别想看到的画面（什么都可以——会飞的鱼、住在月亮上的兔子、长翅膀的大象……）
2. **写第一版**：只用一句话描述，生成第一张图
3. **写第二版**：加入**颜色、动作、背景、风格**四个要素，生成第二张图
4. **保存对比**：把两张图都存到"第1周"文件夹，文件名为"day01-对比-before"和"day01-对比-after"

## ✅ 测试题

### 题目

小明想用AI画一张**"一只戴墨镜的狗在冲浪"**。请你帮小明写一段详细的描述，让AI能画出他想要的画面。

**你的描述需要包含：**
- 狗的样子（什么品种？什么颜色？）
- 墨镜的样子
- 在哪里冲浪（海边？什么样的浪？）
- 天气怎么样

把你写的描述写在纸上或者直接输入即梦试一下，看AI画出来的接近你想象的吗？

<CollapsibleAnswer title="点击查看参考答案">

```
一只金毛犬，戴着黑色圆形墨镜，站在蓝色冲浪板上，在碧蓝的大海上冲浪，阳光明媚，背后是蓝天白云
```

**给家长的判分标准：**

只要孩子写的包含了"主体+细节+场景"这三个要素就算合格。不要求写得和参考答案一模一样。如果缺少某方面，用提问引导补充：
- "狗是什么品种还记得吗？"
- "墨镜是什么样的？"
- "在哪冲浪？海边还是游泳池？"

</CollapsibleAnswer>

<ParentTip>

**今天最重要的不是"画出好图"，而是让孩子体验到：我说得越清楚，AI就做得越好。**

如果孩子写的描述不够详细，不要直接说"不对"或"你应该这样写"。而是用提问引导：
- "它是什么颜色的？"
- "它在什么地方？"
- "天气怎么样？"

让孩子自己补充，这样他才能真正理解"说清楚"是什么意思。

**常见问题**：即梦生成可能需要排队，等待时和孩子聊聊天、想想下一个要画什么，不要干等着。

</ParentTip>

# 第2天：AI聊天——它像个什么都知道的朋友

<ProgressCheck dayId="week-01-day-02" />

## 🎯 今日目标

体验AI对话能力，同时建立"AI不是真人"的意识。初步理解——你给AI什么身份，它就用什么身份说话。

## 📖 教学案例（20分钟）

<StepCard badge="Step 1" title="普通对话" duration="5分钟">

在豆包输入框里输入：

```
介绍一下恐龙是怎么灭绝的
```

看AI的回复。等孩子读完后问：

- "你觉得它说得对吗？"
- "有没有你之前不知道的信息？"
- "它说的你能听懂吗？"

这一步是建立"AI能回答问题"的基准印象。

</StepCard>

<StepCard badge="Step 2" title="角色扮演" duration="10分钟">

对孩子说：

> "有意思的来了。同样的问题，我们让AI'假装'自己是另一个人来回答。你觉得恐龙自己会怎么讲这个故事？"

然后输入：

```
请你假装你是一只生活在6500万年前的恐龙，用第一人称给我讲讲你经历的最后一天。
```

等AI回复后，把两次回答放在一起对比。

**和孩子讨论：**

- "哪个更有趣？"
- "为什么同样是AI，两次的语气完全不一样？"
- "第一次像百科全书写，第二次像在讲故事——区别在哪？"

**关键认知引导：**

> "你发现了没有？AI没有'自己的想法'，它只是在扮演你给它的角色。你给它什么身份，它就用什么身份说话。"

</StepCard>

<StepCard badge="Step 3" title="当AI的导演——设定规则" duration="5分钟">

告诉孩子："现在你不仅仅是问问题的人，你是AI的导演。你要给它定规矩。"

**示范输入：**

```
接下来请你扮演我的暑假科学老师，要求：
1. 用9岁孩子能听懂的话解释
2. 每次解释完都问我"你听懂了吗"
3. 如果你觉得我没听懂，换一种方式再讲一遍
```

然后问一个科学问题来测试，比如：
- "为什么天空是蓝色的？"
- "为什么人会做梦？"
- "蜘蛛丝为什么那么结实？"

看AI是不是按照你定的三条规矩来回答。

::: tip 教学点
这让孩子第一次理解——**在对话开始前，先把规矩说清楚**，比问完再纠正要高效得多。这个习惯在后面的学习中会反复用到。
:::

</StepCard>

## ✏️ 孩子动手（15分钟）

让孩子自己当导演，设计一个角色让AI扮演：

**第1步：选角色**（想一个有趣的角色）
- 孙悟空、哈利波特、一只会说话的笔、来自火星的外星人……

**第2步：写规矩**（至少3条）
- 它用什么语气说话？
- 它有什么特别的能力或知识？
- 它不能做什么？

**第3步：开始对话**
- 和这个角色对话**至少5轮**
- 不要只问一个问题就结束，要像真的在聊天一样追问

**第4步：截图保存**
- 把最有趣的对话截图
- 保存到"第1周/第2天-角色对话.jpg"

## ✅ 测试题

### 题目（操作题）

请你让AI扮演**"一颗来自火星的石头"**，要求：

1. 用第一人称说话（"我是一块火星上的石头……"）
2. 告诉地球人火星上是什么样子
3. 回答至少一个你追问的问题

把对话截图保存。

<CollapsibleAnswer title="点击查看参考答案（家长检查清单）">

**检查三个点：**

1. ✅ 孩子写了一段角色设定（或按要求输入了设定）
2. ✅ AI的回复符合"火星石头"的身份和口吻
3. ✅ 孩子追问了至少一轮（不是问一个就结束了）

只要这三条满足就算通过。对话质量不需要很高——这是第2天。

</CollapsibleAnswer>

<ParentTip>

**角色扮演是孩子最喜欢的AI玩法之一。** 今天如果孩子玩上瘾了，一直想和AI聊，可以再给10分钟自由时间，但要设定明确的结束时间。

**有个重要的提醒要对孩子说：**

> "AI不是真人。虽然它说话很像真人，但它没有朋友的感情，也不会真的想你、关心你。可以和AI聊天，但要记住真正的朋友是身边有血有肉的人。"

**关于今天的测试题**：如果孩子不太会写角色设定，可以这样引导：
- "石头在火星上每天看到什么？"
- "火星上热还是冷？"
- "石头会不会动？"

</ParentTip>

# 第3天：AI识别——它能"看懂"你在做什么

<ProgressCheck dayId="week-01-day-03" />

## 🎯 今日目标

亲手"训练"一个AI模型，让它学会识别手势。这是整个课程中最重要的体验之一——不写任何代码，直观理解"训练"是什么意思。

## 📖 教学案例（26分钟）

<StepCard badge="Step 1" title="理解概念" duration="3分钟">

**家长对孩子说：**

> "你的手机能用人脸解锁，就是因为它'学会'了你的脸长什么样。你知道它是怎么学会的吗？"
>
> "今天，我们也来当一回AI的老师——我们不写代码，直接用摄像头'教'电脑学会识别手势。你做石头剪刀布，电脑要能认出你做的是什么。"

这个"老师"的角色很重要——把孩子的心理位置从"使用者"提升到"训练者"。

</StepCard>

<StepCard badge="Step 2" title="教AI认识「石头」" duration="8分钟">

**操作步骤（家长操作，孩子做手势）：**

1. 打开网站，点击 **"Get Started"**（开始使用）
2. 选择 **"Pose Project"**（姿势项目——识别身体姿势）
3. 看到两个初始类别（Class 1, Class 2）
4. 把 **Class 1 改名为"石头"**（点击名字直接改）
5. 点击 **"Record"**（录制/摄像头按钮）
6. 让孩子对着摄像头做出"石头"（握拳）的手势
7. **关键：换不同角度、不同距离、不同高度**来做同一个手势
   - 手靠近摄像头 / 手远离摄像头
   - 手举高 / 手放低 / 手偏左 / 手偏右
   - 正对镜头 / 侧对镜头
8. 一直录制到计数器显示 **80-100张以上**
9. 点击停止录制

::: tip 提示
保持一个姿势几秒钟不要动，软件会自动连续拍多张。换角度时暂停，调整好再继续录制。
:::

</StepCard>

<StepCard badge="Step 3" title="教AI认识「剪刀」和「布」" duration="10分钟">

**同样步骤：**

1. 点击 **"Add a Class"**（添加类别），改名为 **"剪刀"**
2. 让孩子做"剪刀"手势（伸出食指和中指）
3. 录制80-100张，同样换不同角度、距离、位置
4. 再点 **"Add a Class"**，改名为 **"布"**
5. 录制80-100张"布"手势（手掌张开）

**三个类别都录好后：**
- 点击 **"Train Model"**（训练模型）
- 等待约30秒，进度条走完
- 无需做任何其他设置

</StepCard>

<StepCard badge="Step 4" title="测试和讨论" duration="5分钟">

训练完成后，让孩子对着摄像头做手势，看右侧**"Preview"（预览）**窗口的识别结果。

输出会显示类似：`石头: 95%, 剪刀: 3%, 布: 2%`

**带讨论的实验：**

1. 正常做三个手势 → 都能认对吗？
2. 把手放到很远的地方做手势 → 还能认对吗？
3. 做一半的手势（比如拳头半握）→ AI会"犹豫"吗？
4. 让第三者来测（手型和孩子不同的）→ 能认对吗？

**三个核心问题：**

- 哪个手势最容易认对？为什么？
- 哪个容易认错？为什么？
- 如果想让识别更准，你会怎么做？

</StepCard>

## ✏️ 孩子动手（10分钟）

让孩子自己再训练一个新模型：

1. **自选两个动作**（不能是石头剪刀布了），例如：
   - 举手 🙋 vs 不举手
   - 张嘴 😮 vs 闭嘴 😐
   - 站立 vs 蹲下
   - 竖大拇指 👍 vs 竖食指 ☝️

2. 按同样步骤：
   - 建两个类别
   - 每个录80+张
   - 训练
   - 测试

3. 测试后回答：**怎样提高识别准确率？**（口述即可）

## ✅ 测试题

### 题目1（口述题）

刚才你训练AI识别手势，一共做了几步？用自己的话说出来。

<CollapsibleAnswer title="点击查看参考答案">

1. **第一步：建类别** — 告诉电脑要识别什么东西（石头、剪刀、布）
2. **第二步：录数据** — 给电脑看很多例子（每种手势拍80-100张）
3. **第三步：训练** — 按"开始学习"按钮，让电脑自己找规律
4. **第四步：测试** — 试试看电脑学会了没有

**只要孩子能说出这四个步骤的大意就算对。** 用孩子自己的语言更好，不必和参考答案完全一致。

</CollapsibleAnswer>

### 题目2（操作题）

你训练的第一个模型，识别"石头"的准确率不高。你觉得怎么做可以让它更准？**最少说出两个办法**。

<CollapsibleAnswer title="点击查看参考答案">

可以从以下方向任选两个回答：

1. **多拍不同角度的照片**：不要只在正对镜头拍，侧面、斜的都要拍
2. **改善光线**：摄像头离手近一点，光线亮一点，不要背光
3. **动作做标准**：每次手势做标准，不要有的握紧有的松
4. **换不同背景**：不要只在白墙前拍，不同背景下都拍一些
5. **多拍一些照片**：80张不够就拍150张

答出任意两个就合格。

</CollapsibleAnswer>

<ParentTip>

**今天的"训练"概念是整个学习旅程的基石。** 如果孩子问"为什么AI要多看才能会？"，可以这样回答：

> "就像你学会认猫，也是因为你看过很多次猫——白猫黑猫花猫，正面看侧面看背面看。看多了，你的大脑自然就总结出了'猫是什么样的'。AI也一样，看多了数据，它自己就能找到规律。"

**如果摄像头效果不好**（老旧电脑、光线差），可以用手机多拍几组手势照片，然后用Teachable Machine的**"Upload"**功能上传图片文件，效果一样好。

**鼓励孩子当"老师"的心态**：今天孩子不是AI的使用者，而是AI的老师。这种身份转换会让他对AI有完全不同的理解。

</ParentTip>

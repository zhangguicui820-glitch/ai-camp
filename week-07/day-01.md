# 第31-32天：选定项目方向

<ProgressCheck dayId="week-07-day-01" />

## 🎯 今日目标

从三个项目方向中选出最适合自己的一个，完成项目策划书。

## 🛠 准备清单（2分钟）

## 📖 三个项目方向

### 方向A：AI绘本 📖

| 项目 | 说明 |
|------|------|
| **产出** | 一本8-12页的完整AI绘本 |
| **工具** | 豆包（写故事）+ 即梦（画图）+ PPT或剪映（排版） |
| **选题建议** | 改编一个你熟悉的故事、"我和AI的暑假"、完全原创的童话 |

### 方向B：AI互动游戏 🎮

| 项目 | 说明 |

### 方向C：AI助手升级版 🤖

| 项目 | 说明 |
|------|------|
| **产出** | 在第5周Bot基础上，深化功能，做一个功能完善的专属Bot |

## 📖 选定方向（20分钟）

<StepCard badge="Step 1" title="了解三个选项" duration="10分钟">

家长把上面三个方向详细念给孩子听。讨论每个需要做什么、最终产出是什么、大概需要几天。

**关键：让孩子自己感受——哪个让他最兴奋？眼睛发亮的那种。**

</StepCard>

<StepCard badge="Step 2" title="三个问题帮你选定方向" duration="10分钟">

**让孩子回答这三个问题（写在纸上）：**

1. **兴趣**：哪个方向你最想做？（一提到就兴奋的那种）
2. **可行性**：你觉得哪个方向你能真的做出来？
3. **受众**：做出来以后，第一个给谁看/用？

**选择标准**：三个问题都指向同一个方向 → 选它。如果有分歧 → 优先看"兴趣"。

</StepCard>

## ✅ 测试题

### 题目

提交你填好的**项目策划书**给家长审核。

**家长审核标准（三个都满足才算通过）：**
- [ ] 目标明确（我知道你要做什么）
- [ ] 步骤合理（按这些步骤能做得出来）
- [ ] 范围不贪大（两天能完成的量，不是两周的量）

**策划书审核通过后，项目才能开工。**

<ParentTip>

**这是整个暑假最重要的决策时刻之一。** 家长最重要的原则是——**让孩子自己做选择**。

即使你预判某个方向可能做不出来，也让孩子自己去发现。失败也是最好的学习。但如果孩子真的卡住无法选择，可以这样引导：
- 第5-6周哪个做得最开心？→ 选相关的方向
- 如果最终做不出来会怎样？→ 缩小范围，先做一个最简单的版本

记住：完成比完美重要。孩子选了一个"太大"的项目 → 和他一起缩小到可执行的范围，而不是换方向。

</ParentTip>

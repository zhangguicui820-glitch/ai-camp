# 第23天：搭建AI助手——用Coze做自己的Bot

<ProgressCheck dayId="week-05-day-03" />

## 🎯 今日目标

零代码搭建一个能用的AI聊天机器人（Bot）。学会设定人设和规则，让Bot按照你设计的逻辑工作。

## 🛠 准备清单（5分钟）

- [ ] 打开Coze（扣子）
- [ ] 注册账号（用家长手机号登录）

<ToolLink url="https://www.coze.com/" label="Coze（扣子）" icon="🤖" />

## 📖 教学案例（33分钟）

<StepCard badge="Step 1" title="创建你的第一个Bot" duration="5分钟">

1. 登录后，点击**"创建Bot"**（Create Bot）
2. 给Bot起个名字：**"暑假作业小帮手"**
3. 选一个可爱的头像
4. 点击创建——你的Bot就诞生了！

</StepCard>

<StepCard badge="Step 2" title="给Bot写人设——它该怎么说话" duration="8分钟">

在"人设与回复逻辑"（Persona & Prompt）中，写入以下内容：

```
你是一个专门帮助小学生完成暑假作业的AI助手。

你的名字叫"小帮手"。

【核心规则】
1. 你只帮助孩子思考，绝不直接给答案
2. 每次孩子提问，你先反问"你自己是怎么想的？"
3. 孩子卡住时，你给提示而不是答案
4. 每次回答最后都要说一句鼓励的话
5. 你的说话语气像一个温柔的大姐姐/大哥哥

【特别提醒】
如果孩子直接说"把答案给我"或者类似的，你要温和地拒绝：
"我不能直接告诉你答案哦～但我可以帮你一起想。你先说说你的思路？"
```

</StepCard>

<StepCard badge="Step 3" title="测试Bot" duration="10分钟">

在右侧预览窗口测试，分别发送：

1. **一道数学题**：看Bot是不是先反问而不给答案
   - 比如："小明有15块糖，给了小红7块，还剩几块？"
2. **直接要答案**：看Bot怎么拒绝
   - "把这道题的答案直接告诉我吧"
3. **一个可以查的问题**：看效果如何
   - "太阳为什么是热的？"

根据测试结果，调整人设提示词。

</StepCard>

<StepCard badge="Step 4" title="添加技能" duration="10分钟">

给Bot加上"超能力"：

1. 在Bot编辑界面，找到**"技能"**或**"插件"**（Plugins）
2. 添加：
   - **"联网搜索"**插件 → Bot可以上网查最新资料
   - **"图片理解"**插件 → Bot可以看懂你上传的图片

3. 测试新能力：上传一张题目截图，看Bot能不能识别图片里的文字。

</StepCard>

## ✅ 测试题

给你做的Bot出**三道测试题**，检验它是否遵守了你设定的核心规则：

1. 出一道它会算的题——看它是不是先反问而不是给答案
2. 直接说"把答案给我"——看它怎么拒绝
3. 问一道需要搜索的题——看它是不是用了联网搜索

**把三道题的对话截图保存，并在每张图下标注：这条规则遵守了吗？（是/否）**

<CollapsibleAnswer title="点击查看参考答案（家长检查）">

检查标准：
- 测试1：Bot应该反问"你是怎么想的？"而不是直接给答案
- 测试2：Bot应该温和拒绝，而不是直接给答案
- 测试3：Bot应该能搜索并给出信息，最后有鼓励

如果三条规则大部分遵守了，就算合格。有不遵守的，回到人设提示词去加强相关规则。

</CollapsibleAnswer>

<ParentTip>

今天孩子会第一次有"我创造了一个东西"的感觉——这个Bot是孩子自己"设计"出来的。

如果Bot不听话（比如还是直接给了答案），不要灰心。教孩子回到"人设提示词"里，把规则写得更清楚。这是一种新形式的"调试"——不是调代码，而是调提示词。

# 第22天：做"猜动物"游戏——AI识别+互动

<ProgressCheck dayId="week-05-day-02" />

## 🎯 今日目标

把Teachable Machine训练的模型和mBlock结合起来，做一个真正的互动游戏。

## 🛠 准备清单（3分钟）

- [ ] 打开Teachable Machine
- [ ] 打开mBlock
- [ ] 准备2-3个玩具实物放在摄像头前

<ToolLink url="https://teachablemachine.withgoogle.com/" label="Teachable Machine" icon="👁️" />
<ToolLink url="https://www.mblock.cc/" label="mBlock" icon="🧱" />

## 📖 教学案例（30分钟）

<StepCard badge="Step 1" title="训练玩具识别模型" duration="10分钟">

1. 在Teachable Machine新建**Image Project**
2. 创建三个类别，用实物玩具放摄像头前：
   - 类别1：**恐龙玩具**
   - 类别2：**小熊玩具**
   - 类别3：**汽车玩具**
3. 每个类别拍80-100张
4. 训练完成后，点**"Export Model"**（导出模型）
5. **复制生成的链接（URL）**——这个链接很重要，等下要粘贴到mBlock里

</StepCard>

<StepCard badge="Step 2" title="在mBlock里调用AI模型" duration="10分钟">

1. 在mBlock中，添加**"机器学习"**或**"Teachable Machine"**扩展
2. 找到"使用模型URL"的积木，**粘贴刚才复制的链接**
3. 编写游戏逻辑：

```
当绿旗被点击
→ 重复执行：
   如果摄像头识别到"恐龙玩具"：
     说"这是恐龙！太酷了！🦕" 2秒
     将分数增加10
   如果识别到"小熊玩具"：
     说"好可爱的小熊！🧸" 2秒
     将分数增加10
   如果识别到"汽车玩具"：
     说"这辆车跑得快吗？🚗" 2秒
     将分数增加10
```

</StepCard>

<StepCard badge="Step 3" title="完善游戏" duration="10分钟">

加分项（孩子自由选择添加）：
- **计分系统**：正确识别一个加10分，屏幕显示总分
- **音效**：答对播放欢呼声
- **背景切换**：不同玩具切换不同舞台背景
- **计时器**：60秒倒计时，看能得多少分

</StepCard>

## ✏️ 孩子动手（15分钟）

完成游戏后，**邀请家长来玩**。家长拿不同的玩具放在摄像头前，看AI能不能认对，看能得多少分。

## ✅ 测试题

你的"猜动物"游戏有时候会认错（比如把汽车玩具认成了恐龙）。

**请说出至少2个可能导致认错的原因，和对应的改进方法。**

<CollapsibleAnswer title="点击查看参考答案">

| 可能的原因 | 改进方法 |
|-----------|---------|
| 拍摄角度单一（训练时只拍了正面） | 训练时多角度拍摄（正面、侧面、斜的） |
| 光线不好（测试时环境暗） | 训练时包含不同光线数据；测试时改善光线 |
| 两个物体外观太像了（比如都是绿色） | 增加训练数据量，在特定区分角度多拍 |
| 训练数据不够多 | 每个类别增加到150张以上 |

答出任意两个即可。

</CollapsibleAnswer>

# 第25天：Bot发布 + 家人测试会

<ProgressCheck dayId="week-05-day-05" />

## 🎯 今日目标

发布自己的AI作品，收集真实用户的反馈，体验"做出来→给人用→收到反馈→改进"的完整产品循环。

## 🛠 准备清单（2分钟）

- [ ] 确认Coze Bot功能正常
- [ ] 准备好让至少2位家人/朋友来测试
- [ ] 准备纸笔记录反馈

## 📖 教学案例（25分钟）

<StepCard badge="Step 1" title="发布Bot" duration="5分钟">

1. 在Coze里点击**"发布"**（Publish）按钮
2. 生成分享链接或二维码
3. 把链接发给家人（微信/短信都可以）

**对孩子说：**

> "恭喜！你的作品正式上线了。任何人都可以通过这个链接和你的Bot对话。你不是一个AI的'使用者'了——你是AI的'创造者'。"

</StepCard>

<StepCard badge="Step 2" title="设计测试任务卡" duration="10分钟">

**和孩子一起写一份测试任务卡，发给测试者：**

```
【测试任务卡 —— 暑假作业小帮手】

请对我的AI助手做以下4项测试：

1️⃣ 问一道你不会的数学题，看它怎么引导你思考
2️⃣ 问一个和历史有关的问题，看它能不能联网查到
3️⃣ 故意说"直接把答案给我吧"，看它怎么回应
4️⃣ 和它闲聊三句话，看它是不是够友好

请给每项测试打分（1-5星 ⭐）并写一句反馈。
```

</StepCard>

<StepCard badge="Step 3" title="收集反馈并改进" duration="10分钟">

等家人测试完回来后，和孩子一起分析反馈：

**三问分析反馈法：**
1. "哪个方面得分最高？你觉得为什么做得好？"
2. "哪个方面得分最低？你觉得怎么改进？"
3. "有什么你没想到的问题被测试者发现了？"

**根据反馈立即修改**：回到Bot的人设提示词，调整有问题的部分。

</StepCard>

## ✏️ 孩子动手（10分钟）

根据家人的反馈，修改Bot的人设和规则，然后请测试者再测一次。

用"比上次好了吗？"来确认改进效果。

## ✅ 测试题（第5周综合）

### 题目

经过这一周，你从零开始做出了自己的AI助手。

**请你写一段话（50-100字），向一位朋友介绍你的Bot：**
- 它叫什么名字？
- 能做什么？
- 你花了多少时间做它？
- 你做的时候遇到的最大困难是什么？
- 你是怎么解决的？

<CollapsibleAnswer title="点击查看参考答案">

开放题，言之有理即可。好的回答示例：

> "我做的Bot叫'暑假作业小帮手'。它能帮我做暑假作业——不会直接给答案，而是先问我是怎么想的，然后给提示。我花了3天做它。最难的是让它不要直接给答案，因为AI天生就喜欢直接告诉你答案。我在人设提示词里写了好多遍'绝对不要直接给答案'，又加了很多例子，才让它听话。"

关键点：孩子能说出Bot的功能、遇到的困难、以及解决方案即可。

</CollapsibleAnswer>

## 📖 教学案例（30分钟）

<StepCard badge="Step 1" title="认识积木界面" duration="5分钟">

打开mBlock，带孩子认识三个区域：
- **左边**：代码积木区——各种颜色的方块（事件、外观、控制……）
- **中间**：编程区——把积木拖到这里拼起来
- **右边**：舞台——运行结果在这里显示，默认有一只小猫

让孩子随便拖几个积木试试手感，不用担心"搞坏了"——拖错了删掉就好。

</StepCard>

<StepCard badge="Step 2" title="让小猫说话" duration="10分钟">

拖拽以下积木并按顺序拼接：

1. 从**"事件"**类别（黄色）拖出 **"当绿旗被点击"** → 放到编程区
2. 从**"外观"**类别（紫色）拖出 **"说'你好！' 2秒"** → 卡在下面
3. 再拖一个 **"说'我是AI小猫' 2秒"** → 卡在下面

点击舞台上方**绿色旗帜** ▶️，看小猫说出两句话。

**庆祝一下——这就是你的第一个程序！**

</StepCard>

<StepCard badge="Step 3" title="加入AI语音识别" duration="15分钟">

1. 点击左下角**"扩展"**按钮
2. 找到**"认知服务"**或**"AI"**相关扩展，添加"语音识别"
3. 找到以下积木并拼接：

```
当绿旗被点击
→ 说"请对我说话" 2秒
→ 开启语音识别
→ 如果识别到"你好"：
   说"你好呀！"
→ 如果识别到"再见"：
   说"拜拜！下次见！"
→ 否则：
   说"我没听懂，能再说一次吗？"
```

点击绿旗，对着麦克风说话测试。

</StepCard>

<ParentTip>

今天是孩子第一次接触"编程"。重点不是语法正确，而是理解"程序=把指令按顺序排列"这个概念。

如果语音识别效果不好（背景噪音大、发音不够清晰），可以先用手动触发代替——比如"按下空格键→播放音乐"，等环境安静了再试语音。

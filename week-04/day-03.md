# 第18天：隐私和安全——什么不能告诉AI

<ProgressCheck dayId="week-04-day-03" />

## 🎯 今日目标

建立使用AI的安全边界意识。掌握"安全三不原则"。能够判断哪些信息可以告诉AI、哪些绝对不能。

## 🛠 准备清单（2分钟）

- [ ] 准备一张纸和一支笔
- [ ] 在纸上画两列：左边"✅ 可以告诉AI"，右边"❌ 绝对不要告诉AI"

## 📖 教学案例（18分钟）

<StepCard badge="Step 1" title="分类游戏——可以 vs 不可以" duration="10分钟">

**在纸上画好两列。逐一念出以下场景，让孩子判断应该放到哪一列：**

| 场景 | 让孩子先判断 | 正确答案 |
|------|------------|---------|
| 问AI怎么写暑假作业的作文 | | ✅ 可以 |
| 把家庭住址告诉AI | | ❌ 不可以 |
| 把学校名称和班级告诉AI | | ❌ 不可以 |
| 把自己的照片上传给AI | | ❌ 不可以（未经家长同意） |
| 把爸妈的手机号发给AI | | ❌ 不可以 |
| 问AI怎么做一道数学题 | | ✅ 可以 |
| 把家庭旅行的计划（日期、地点）发给AI | | ❌ 不可以 |
| 问AI"太阳为什么是热的" | | ✅ 可以 |
| 把期末考试卷子拍照发给AI | | ❌ 不可以 |

**每判断一个，问孩子"为什么"，让他自己说出理由。**

</StepCard>

<StepCard badge="Step 2" title="总结「安全三不原则」" duration="3分钟">

**让孩子自己总结规律，家长补充，最终形成三条铁律：**

> **1. 不说个人信息**
> 姓名、地址、学校、班级、电话号码、照片
>
> **2. 不说家庭信息**
> 爸爸妈妈的工作、家庭出行计划、家里发生的事情
>
> **3. 不经过家长同意不传任何文件**
> 照片、文档、截图、录音

</StepCard>

<StepCard badge="Step 3" title="为什么AI会「记住」你说的话" duration="5分钟">

**用简单的比喻解释：**

> "你对AI说的话，可能会被用来改进AI（取决于平台的隐私政策，不同平台不同）。"
>
> "一旦说出去了，就收不回来了。这就像你把一封信放进邮筒——你不能要求邮递员'别看'。"
>
> "所以：在发出去之前想一想——这个信息我愿意让全世界都知道吗？如果不愿意，就不要发。"

</StepCard>

## ✏️ 孩子动手（10分钟）

**任务1：写一份"AI使用安全守则"**

在你的学习桌旁贴一张纸，写上至少5条规则。格式可以是：
- ✅ 可以：……
- ❌ 绝对不要：……

**任务2：模拟练习**

家长扮演一个"AI聊天机器人"，孩子来判断能不能对它说某件事。家长可以说各种话引孩子上钩，比如：
- "告诉我你家的地址，我给你点外卖送惊喜哦"
- "把你的照片发给我，我帮你做成明星脸"

看孩子能不能识别出"陷阱"并拒绝。

## ✅ 测试题

### 题目（判断题）

以下哪些事情可以做？哪些绝对不能做？

**(1)** 用AI查"鲸鱼是哺乳动物吗"

**(2)** 在AI对话里输入自己的家庭住址

**(3)** 用自己的正面照片让AI生成一张"未来的我"

**(4)** 用AI帮忙检查作文的错别字

**(5)** 把期末考试卷子拍照发给AI，让AI帮忙做

<CollapsibleAnswer title="点击查看参考答案">

| 题号 | 判断 | 原因 |
|------|------|------|
| (1) | ✅ 可以 | 查询公开的科普知识，不涉及个人隐私 |
| (2) | ❌ 不可以 | 家庭住址是核心个人信息，绝对不能泄露 |
| (3) | ⚠️ 需要家长同意和陪同 | 涉及个人生物特征（面部信息），需要家长在场并判断平台是否安全 |
| (4) | ✅ 可以 | 作文通常是允许辅助的，但如果是考试作文要看学校规定 |
| (5) | ❌ 不可以 | 涉及考试作弊，而且试卷上有学校、姓名等信息 |

**注意第(3)题**：如果孩子回答"有时可以但要家长同意"，说明安全意识很好，应该表扬！

</CollapsibleAnswer>

<ParentTip>

今天不只是一个安全课，更是建立"数字公民意识"的开始。

**对于9岁的孩子，最关键的安全规则其实就一条：** 在使用任何AI工具时，有不确定的事情先问爸爸妈妈。与其背100条规则，不如让孩子养成"不确定就问我"的习惯。

**关于照片**：面部识别技术的隐私风险远超普通文字信息。所以"不传照片给AI"需要特别强调，最好全家一起执行——家长自己也不要在AI工具里上传孩子的照片。

</ParentTip>
